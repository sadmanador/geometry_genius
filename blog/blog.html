DOM or Document object model is the way of browser accessing the html in element in browser engine. DOM make every element of a html file branches of document root. every single branch can be accessed with node branch selection method.

There many ways of selecting node elements:
  1. childNode and parentNode method
  2. with querySelector and querySelectorAll
  3. getElementsByClassName, getElementsByTagName, getElementsById
  4. using direct event like: onclick, onmouseover, etc..

Any event in the DOM tree go from the target element to the root of the DOM. This tendency of DOM element events can be very useful in some cases, yet problematic in others. The propagation of the event can be controlled to passover into its parent nodes.

When a function is passed into another function minimization and reusing that nested function in similar scenario to avoid duplication of code is called callback function(The nested one). even in some by default cases callback functions are use like: event, Math, fetching data and many more.
